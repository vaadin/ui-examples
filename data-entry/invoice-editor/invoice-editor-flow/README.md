# Invoice Editor UI Example

Import this project to the IDE of your choice as a Maven project. You need to have Java 8 installed.

Run using `mvn jetty:run` and open [http://localhost:8080](http://localhost:8080) in browser.

If you are new to Vaadin, we recommend our [getting started tutorial](https://vaadin.com/tutorials/getting-started-with-flow).
